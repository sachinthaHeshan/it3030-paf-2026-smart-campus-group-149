## Deploying the frontend to GitHub Pages

The workflow **Deploy frontend to GitHub Pages** will fail with **404** until GitHub Pages is enabled (one-time setup):

1. Open: **https://github.com/sachinthaHeshan/it3030-paf-2026-smart-campus-group/settings/pages**
2. Under **Build and deployment**, set **Source** to **GitHub Actions**.
3. Save. Then re-run the workflow or push a new commit.

After that, the site will be at: **https://sachinthaheshan.github.io/it3030-paf-2026-smart-campus-group/**
